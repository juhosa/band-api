# BandAPI API

One endpoint returning an array of bands.

## `GET /get-bands`

Returns

```json
[
  { "id": 0, "name": "Daft Punk", "genre": "techno", "membercount": 2 },
  { "id": 1, "name": "Metallica", "genre": "metalli", "membercount": 4 },
  { "id": 2, "name": "Kiss", "genre": "rock" }
]
```
